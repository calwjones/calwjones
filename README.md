## About Me

hii im callum — a final year compsci student at UWE Bristol with a focus on systems programming, game development, and full-stack applications. Currently building a 2D game engine from scratch in C++ for my final project.

- 🔭 Currently working on a custom 2D game engine with SFML & Dear ImGui
- 🎮 Built a Tetris clone with sand physics optimised for 60fps
- 🧮 Created a GUI calculator with a custom expression parser using the Shunting-yard algorithm
- 🎥 Building a Movie Backlogger with 2 person match swiper and roulette wheel
- 🌱 Always deepening my skills in systems programming & low-level development
- 📍 Based in **Bristol, UK**

### 🎮 [2D Game Engine](https://github.com/calwjones/GameEngine)
Custom 2D game engine built from scratch in C++ with SFML for rendering and Dear ImGui for the editor UI. Features a fixed timestep game loop, entity management, AABB collision detection, visual level editor with drag-and-drop, and JSON-based level serialization.

### 🧱 [Tetris with Sand Physics](https://github.com/calwjones/Tetris)
Tetris clone with two modes: Classic and Sand Physics. Sand mode turns pieces into particles with gravity-based spreading and path-based line clears. Optimised for 60fps with ~3,200 sand grains using typed arrays and precomputed trig tables.
